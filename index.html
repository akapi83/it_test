About me

防水工事会社の社長をしながらWEBデザイナーをしている一風変わった経歴の持ち主です。

卒業後、韓国語通訳を経て防水工事の会社を設立、現場での施工と共に経理・法務・勤怠 管理・施工管理等の管理業務、経営管理業務を並行して行ってきました。 2009 年よりチラシ・パンフレットのデザインを経験、2018 年から WEB サイト制作の請負を しており、更に技術を深めるため JavaScript 及び React を勉強しています。

私にとってのWEB

父が広告代理店でデザイナーとして仕事をしていた関係で、私は7歳からパソコンに触れていました。PhotoshopやIllustratorでデザインしていく楽しさを感じつつ、インターネットの普及により様々な情報をWEBサイトから入手するようになり、自分で作って見たいと思うようになりました。

動画編集の技術と、WEBサイト制作を組み合わせてより効果的に情報を提供できるWEBサイトを作りたいと常に思っています。

スマートフォンの普及が進み、今後5Gが始まればインターネットを通した情報伝達は更に発展すると思われますが、そこには技術の進化についていけず、取り残される方々が出てくると思います。WEBサイトを制作し情報を発信できる技術を幸いにも得られた私は、そのような方々のためにWEBサイトを作り、情報提供の場を提供し、彼らの力になりたいと思っています。
